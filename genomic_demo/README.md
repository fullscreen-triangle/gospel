## Quick Setup

1. **Create virtual environment** (recommended):

   ```bash
   cd genomic_demo
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

2. **Install minimal dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
